# Book E-commerce Website

A full-stack e-commerce website for selling books online.

## Project Overview

This project allows users to browse books, add products to cart, place orders, and manage their accounts.

## Technologies

### Frontend

* ReactJS
* Axios
* React Router
* Bootstrap

### Backend

* Spring Boot
* Spring Security
* Spring Data JPA
* MySQL

## Features

### User Features

* Register/Login
* Browse books
* Search books
* Add to cart
* Place orders
* View order history

### Admin Features

* Manage books
* Manage categories
* Manage orders
* Manage users
